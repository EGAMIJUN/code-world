# PRD — 戦闘車両（戦車に乗れる / 車に乗りながら撃てる）

`/world`（`apps/web/src/app/world/ThreeWorld.tsx`）への機能追加。要件すり合わせ（14問）の結論をまとめた要件定義書。**この文書を承認後に実装着手**。

---

## 1. 目的・ゴール

- **車に乗りながら銃を撃てる**：運転（WASD）しながらマウスで自由に狙って撃つ「ドライブバイ」。
- **戦車に乗れる**：装甲が厚く、専用の主砲（AOE砲撃）を撃てる重量級の乗り物。戦車では主砲と手持ち銃を切り替え可能。
- 既存の徒歩・射撃・ナイフ・既存の車・通常モードの敵を壊さない。パフォーマンス維持。

---

## 2. 用語（簡単な補足）

| 用語 | 意味 |
|---|---|
| フリーエイム | 車の向きと独立して、マウスで照準を動かせる方式（GTA/BF系） |
| 砲塔（turret） | 戦車上部の、砲身ごと回る部分。マウス照準に追従して回る |
| AOE / 範囲ダメージ | 着弾点の周囲にも及ぶダメージ |
| ヒットスキャン | 撃った瞬間に直線上の的へ即命中（弾の飛行時間ゼロ） |
| クールダウン | 次に撃てるまでの待ち時間 |
| ADS | 右クリックで覗き込むズーム照準 |
| ドライブバイ | 走行しながらの射撃 |

## 4. 機能仕様

### 共通基盤（PR①で構築、PR②が乗る）

**4.1 フリーエイム運転カメラ**
- 三人称チェイスカメラは維持しつつ、運転中の**マウス入力を「照準ヨー/ピッチ」として復活**させる（現状は運転中マウス無効）。
- カメラは照準方向を追って車の周囲をオービット。撃つ方向＝画面中央のクロスヘアからのレイ（徒歩の射撃と同じ「画面中央レイ」方式を流用）。
- 左クリック（モバイルFIRE）＝現在の武器を発射。運転中ADSは無効。

**4.2 乗車中の射撃（手持ち銃）= 「車で撃てる」**
- 現在の `fire()` / `meleeAttack()` の `if (drivingRef.current) return` ガードを撤去し、運転中も発砲可能に。
- 弾の発射原点は車両位置（≒プレイヤーの focalPoint）。ダメージ/弾数/リロード/反動は徒歩と同一。
- WASDで走りつつマウスで照準して同時に撃てる（ドライブバイ）。

**4.3 乗り物HP・破壊・乗員ダメージ**
- `Vehicle` に `hp` / `maxHp` を追加。被弾（敵弾・爆発・グレネード）で減少。
- **ダメージは乗り物HPのみが受ける**（中のプレイヤーは無傷＝Q10）。
- HP=0で**爆発**（既存 `spawnExplosion` を大きめに流用）＋乗り物メッシュ撤去/残骸化。
- プレイヤーは車両の脇に**強制降車＋大ダメージ**（例：残りHPをわずかに）。爆発のAOEに巻き込まれる扱い。
- 車（既存）は脆い（小HP・全ダメージ）。

### PR②：戦車

**4.4 戦車エンティティ**
- 既存 `Vehicle` システムを拡張（`kind: "car" | "tank"`）。`makeVehicle` に戦車モデル（車体＝ハル＋回転砲塔＋砲身、キャタピラ風の見た目）を追加。
- 新規で2〜3基スポーン（大通り沿い等、`findSafeSpawnNear` で安全配置）。既存の障害物戦車（MAP_OBJECTS type 3）は触らない。
- 当たり判定半径は車より大きい。装甲＝小銃ダメージを大幅軽減、爆発物に弱い（Q8）。HPは大。

**4.5 主砲（キャノン）**
- フリーエイムの照準方向に、**遅い砲弾**を発射（既存 `isGrenade` 投擲物理を流用：飛行→着弾/壁衝突→AOE爆発）。
- 弾数無限、**クールダウンで連射制限**。直撃でなくても範囲ダメージ。
- 砲塔がマウス照準（ヨー）に追従して回転、砲身はピッチで上下（屋上の敵も狙える）。

**4.6 戦車での武器切替**
- 乗車時は**主砲が初期選択**。`1/2/3`で手持ち銃、専用キーで主砲に戻る（既存の武器スロットUIに主砲を追加）。
- モバイルは武器切替ボタンで主砲⇄手持ちをトグル。

## 5. 操作まとめ

### PC
| 入力 | 動作 |
|---|---|
| `E` | 乗降（トグル、既存） |
| `WASD` | 車体の操舵（前後＋旋回） |
| マウス | 照準（フリーエイム）＝カメラ/砲塔が追従 |
| 左クリック | 現在の武器を発射（手持ち or 主砲） |
| `1/2/3` | 手持ち銃に切替（戦車のみ） |
| 主砲キー（要確認：`4` or `V`） | 主砲に切替（戦車のみ） |
| `R` | リロード（手持ち、走行中も可） |
| 右クリック（ADS） | 運転中は**無効** |

### モバイル
- 左スティック＝操舵 / 右スティック＝照準 / FIREボタン＝発射 / 武器切替ボタン＝主砲⇄手持ち / 乗降ボタン（既存）/ ADSボタンは運転中非表示。

## 6. 非対象（今回やらないこと）

- **轢き（run-over）**：次PR。
- **PvP（FFA/TDM）での乗り物**：今回はPvPに乗り物を出さない。同期も別PR。
- **敵AI/ボットの乗車・降車AI・敵戦車**：プレイヤー専用。NEXT_STEPSの将来項目。
- **壁・建物の破壊**：破壊対象は乗り物と敵のみ。マップ構造物は従来通り不可。
- 完全な車両物理（サスペンション等）・3D完全同期ネットコード。

## 7. 要確認のデフォルト前提（承認 or 修正をお願いします）

実装前に以下を既定値として進める想定です。違和感あれば指摘ください。

1. **既存の車もPvPでは非表示にする**（半同期の戦闘車両を避けるため。現状PvPに出ている3台の車を非PvPモード限定へ）。
2. **主砲キー＝`4`**（現在ナイフが`4`。戦車搭乗中は手持ちスロットを `1/2/3` に寄せ、`4`を主砲に再割当て。徒歩時の`4`=ナイフは不変）。※代替：`V`や`F`。
3. **主砲クールダウン ≈ 2.5秒**、AOE半径・ダメージはグレネードを基準に強化（例：半径6m・最大140ダメージ）。**至近自爆防止の最小射程**あり。砲弾は自分の乗り物・自分には当たらない。
4. **HP目安**（後で調整）：戦車 ≈ 2000、車 ≈ 250。小銃の対戦車ダメージは大幅減衰、爆発物は等倍以上。
5. **破壊演出**：大きめの爆発＋短い煙、メッシュは数秒で除去（または黒焦げ残骸）。
6. **降車時**は武器が手持ちに戻る。乗車後の初期武器は車＝直前の手持ち、戦車＝主砲。
7. **敵の被ダメ**：敵は従来通りプレイヤー（focalPoint）を狙うが、命中分は乗り物HPに吸われる（Q10）。敵グレネード/ヘビーの爆発が戦車に有効な対抗手段。

## 8. 実装メモ（流用する既存システム）

- 乗り物：`Vehicle` interface / `makeVehicle` / `spawnVehicle` / `enterVehicle` / `exitVehicle` / `updateVehicle`（`drivingRef`）。
- 砲弾AOE：`isGrenade` 弾＋`detonateGrenade()`（飛行・重力・着弾/壁/信管・範囲ダメージ）。`fromEnemy` 同様にダメージ対象を制御。
- 射撃：`fire()`（画面中央レイ＋`wallMeshes`遮蔽）、`applyEnemyKill()`、`spawnExplosion()`。
- 破壊判定：`collidesWithWall` / `pointInsideWall` / `ALL_AABBS`。
- カメラ：`updateVehicle` の三人称チェイス（バグ修正済みの壁めり込み回避ロジック）に照準オービットを追加。
- モバイル：既存の左右スティック＋ボタン群、乗降ボタン（`vehicleActionRef`）。

## 9. 受け入れ条件（Doneの定義）

- PR①：車に乗り、WASD走行しながらマウス照準で手持ち銃を撃てる。敵弾で車HPが減り、0で爆発→放り出されて大ダメージ。徒歩/既存射撃/ナイフ/既存運転が回帰しない。`build`/`lint`/`tsc` 全通過。
- PR②：戦車に乗れる。主砲（AOE砲撃・クールダウン）と手持ち銃を切替。小銃を弾き爆発物に弱い装甲。砲塔がマウスに追従。遅く重い挙動。PR①同様に破壊・回帰なし・全チェック通過。
- 両PR：パフォーマンス低下なし。モバイルでも乗車射撃が可能。
